Dette prosjektet er en enkel plattformspill-prototype laget med Pygame. Spillet viser grunnleggende mekanikker som spillerkontroll, hopping, tyngdekraft og kollisjon med plattformer.


Hva jeg har tenkt som et slutt mål!
Spilleren kan bevege seg til venstre og høyre, hoppe og angripe fiender med et våpen som returnerer. Det finnes også flere typer fiender: små minions som går frem og tilbake på plattformene, og en boss som dukker opp etter at minions er eliminert.

Koden inneholder separate klasser for spiller, våpen, plattformer, minions og boss, og styrer både bevegelse, kollisjon og helse. Denne versjonen er ment som en prototype for videre utvikling av historie, animasjon og flere nivåer.
